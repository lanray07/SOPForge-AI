# SOPForge AI App Store Asset Pack

This folder contains upload-ready App Store Connect assets for the SOPForge AI iOS listing.

## Upload Order

1. App icon: `AppIcon/app-icon-1024.png`
2. iPhone screenshots: `Screenshots/iPhone_6_9/*.png`
3. Optional iPhone 6.5-inch screenshots: `Screenshots/iPhone_6_5/*.png`
4. iPad screenshots: `Screenshots/iPad_13/*.png`
5. Subscription metadata: `Metadata/subscription_metadata.md`
6. Subscription review screenshots: `Subscriptions/ReviewScreenshots/*.png`
7. Optional promoted subscription images: `Subscriptions/PromotionalImages/*.png`
8. Listing copy: `Metadata/app_store_metadata.md`
9. Review notes: `Metadata/review_notes.md`
10. Privacy policy: `Metadata/privacy_policy.md`

## Screenshot Sizes

Apple accepts one to ten screenshots per required display family. This pack includes five screenshots for each required universal-app display:

- iPhone 6.9-inch: 1290 x 2796 px
- iPhone 6.5-inch: 1242 x 2688 px
- iPad 13-inch: 2048 x 2732 px
- Subscription promotional images: 1024 x 1024 px

## Notes

- The screenshots are high-resolution product-page renders based on the implemented SwiftUI screens and copy.
- Subscription review screenshots show the in-app paywall offer and are for App Review only.
- Subscription promotional images are optional and intended for promoted In-App Purchases/subscriptions.
- Replace the placeholder support, marketing, backend, and privacy URLs with final public URLs before submission if you do not want to use the GitHub repository links.
- App previews are optional and not included.
