# SOPForge AI App Store Asset Pack

This folder contains upload-ready App Store Connect assets for the SOPForge AI iOS listing.

## Upload Order

1. App icon: `AppIcon/app-icon-1024.png`
2. iPhone screenshots: `Screenshots/iPhone_6_9/*.png`
3. iPad screenshots: `Screenshots/iPad_13/*.png`
4. Listing copy: `Metadata/app_store_metadata.md`
5. Review notes: `Metadata/review_notes.md`
6. Privacy policy: `Metadata/privacy_policy.md`

## Screenshot Sizes

Apple accepts one to ten screenshots per required display family. This pack includes five screenshots for each required universal-app display:

- iPhone 6.9-inch: 1290 x 2796 px
- iPad 13-inch: 2048 x 2732 px

## Notes

- The screenshots are high-resolution product-page renders based on the implemented SwiftUI screens and copy.
- Replace the placeholder support, marketing, backend, and privacy URLs with final public URLs before submission if you do not want to use the GitHub repository links.
- App previews are optional and not included.
